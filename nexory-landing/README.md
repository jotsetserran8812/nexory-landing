# Nexory — Landing Page

Landing page oficial de [Nexory](https://nexory.io) — Agentes de IA para atención al cliente 24/7.

## Deploy

Este proyecto se despliega automáticamente en Netlify desde la rama `main`.

## Estructura

```
nexory-landing/
├── index.html       # Landing page principal
├── assets/          # Logos e imágenes de marca
├── uploads/         # Imágenes adicionales
└── netlify.toml     # Configuración de Netlify
```
